How It Works

Every entry follows the same transparent format so comparisons are meaningful.

1

Prompt

A text prompt (or scene description) is shared across all entries in a comparison group.

2

Generate

Run the prompt through your chosen LLM model + 3D/generative software pipeline.

3

Record

Capture the output as a video. Upload to YouTube and note the exact model and software versions used.

4

Submit

Open a Pull Request adding your video card to the gallery. The community reviews and merges it.
